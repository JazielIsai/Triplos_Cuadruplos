Código Intermedio

El código intermedio puede pensarse como un programa para una máquina abstracta, en el que las variables son posiciones de memoria. Existen maneras formales para representar código intermedio. Estas notaciones simplifican la traducción de nuestro código fuente a nuestro código objeto ya que ahorran y acotan símbolos de la tabla de símbolos.
